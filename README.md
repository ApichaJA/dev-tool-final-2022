# วิธีติดตั้งเพื่อใช้งาน
### 1. เข้าไปยังโปรเจค
```
cd dev-tool-final-2022
```
### 2. ขั้นตอนแรก Install Dependencies
```
yarn install 
```
### 3. รันคำสั่ง เพื่อเปิดการทำงาน Frontend
```
yarn serve
```


# วิธีเปิดใช้ Docker

```
docker-compose up -d --build
```
